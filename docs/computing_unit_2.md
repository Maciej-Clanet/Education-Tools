# Unit Brief

Learners study computer-system fundamentals: hardware/software roles, component interaction, data use.

# Unit Intro

Computing pros need know how/why components + data behave. Support uses system-integration knowledge for fault finding + fixes. Programmers use computer-operation knowledge for efficient software.

Unit covers hardware/software relationship, components alone/together, storage/processing, data transmission/use, impact on orgs/individuals. Foundation for HE/work.

# Assessment

Pearson written exam. 1h45. 80 marks. Availability Jan + May/June. First May/June 2017.

# Assessment outcomes

**AO1** Know/understand facts, terms, standards, concepts, processes. Cmd: complete/draw/give/identify/name/state. 1-5 marks.

**AO2** Apply knowledge to real scenarios. Cmd: calculate/complete/demonstrate/describe/draw/explain/produce. 1-5 marks.

**AO3** Select/use tech + procedures to explore outcomes/find contextual solutions. Cmd: calculate/demonstrate/develop/explain/produce. 1-6 marks.

**AO4** Analyse/evaluate data/info/tech/procedures to recommend + justify solutions. Cmd: analyse/demonstrate/discuss/produce/write. 6-12 marks.

**AO5** Connect technologies/procedures/outcomes/solutions to resolve computing problems. Cmd: evaluate/produce/write. 6-12 marks.
