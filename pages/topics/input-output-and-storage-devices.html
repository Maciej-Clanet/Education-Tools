Where this appears

These ideas come up in office equipment, accessibility tools, gaming hardware, industrial sensors, media systems, and data storage solutions across the whole unit.

Input

Input devices capture data or instructions for the system

Input devices allow people or environments to send information into a computer system. The right choice depends on how the data is created and how accurate, fast, or accessible the method must be.

Text and commands

Keyboard, keypad, barcode reader, or card reader.

Position and selection

Mouse, trackpad, touchscreen, stylus, or game controller.

Real-world signals

Microphone, camera, scanner, thermometer, light sensor, or motion sensor.

Keyboard Good for accurate text entry and shortcuts, but slower for drawing or gesture-based interaction.
Mouse or trackpad Useful for pointer control in graphical interfaces and selecting on-screen objects.
Touchscreen Lets users interact directly with the display, which is useful in mobile devices and kiosks.
Microphone Captures sound input for calls, voice control, or recording.
Scanner or camera Captures images or physical documents as digital data.
Sensor Collects environmental data such as temperature, light, or motion in automated systems.

Output

Output devices present results, feedback, or media to the user

Output devices matter because they shape the user experience. Clarity, speed, accessibility, and quality are often the key decision points.

Visual output

Monitors, projectors, status displays, and printer output.

Audio output

Speakers, headphones, alarms, and screen-reader speech.

Physical feedback

Printed pages, haptic vibration, and control signals to actuators.

Output device Main use Strength Trade-off
Monitor Visual display Immediate and detailed feedback Needs visual attention and screen space
Printer Physical hard copy Permanent paper output Uses consumables and is slower than display output
Speakers or headphones Audio output Useful for alerts, media, and accessibility Can disturb others or be missed in noisy places
Projector Large shared display Good for group viewing Usually lower contrast than a close-up monitor

Accessibility matters here too

Output choice can affect accessibility. A screen reader needs audio output, a user with low vision may need a high-contrast display, and a busy public space may need large visual prompts instead of quiet sounds.

Storage

Storage devices keep data for later use

Storage choice depends on speed, capacity, portability, resilience, and cost. In exam answers, it helps to compare the storage feature directly with the scenario need.

Magnetic storage

Hard disk drives use spinning platters and moving read/write heads.

Best fit: High capacity at lower cost per gigabyte.

Flash storage

SSDs and USB drives store charge in memory cells with no moving parts.

Best fit: Fast access, portability, and shock resistance.

Optical storage

Discs use marks on a reflective layer read by a laser.

Best fit: Some media distribution or low-cost archiving cases.
SSD fast and durable HDD high capacity Optical removable media Flash drive portable
SSD Fast access, no moving parts, and strong for portability, but usually more expensive per gigabyte than hard drives.
HDD Offers large capacity at lower cost, but is slower and more vulnerable to physical shock.
USB flash drive Removable and easy to carry, useful for transfer, but easier to lose and not ideal as the only backup method.
Optical disc Useful in some archive or media distribution situations, but offers lower speed and less convenience than modern flash storage.

Worked judgement

A travel photographer may prefer a fast external SSD for editing on the move, while a school archive may favour a high-capacity HDD system because price per terabyte matters more than top speed.

Storage Technology

Magnetic storage and flash memory store bits in different ways

The technology inside a storage device affects speed, reliability, cost, noise, power use, and how suitable it is for a given system.

Magnetic hard disk

A hard disk drive stores data on circular platters coated in magnetic material. The platters spin while a read/write head moves to the correct track and changes or detects magnetic patterns that represent bits.

Flash memory

Flash storage keeps data by trapping electrical charge in memory cells. Because there are no spinning platters or moving heads, an SSD can access data quickly and is less vulnerable to knocks while in use.

Feature Magnetic HDD Flash SSD or USB storage
Moving parts Yes, spinning platters and moving heads. No moving mechanical parts.
Access time Limited by seek time and rotational delay. Very low because cells can be addressed electronically.
Capacity and cost Often strong for very large, lower-cost storage. Usually faster, but often higher cost per gigabyte.
Durability in portable use More vulnerable to shock while spinning. Better for movement because there are no moving parts.

Exam wording to use

Avoid saying only "SSD is faster". Explain why: an SSD has no moving read/write head, so it does not wait for a platter to spin to the right position before accessing data.

HDD Performance

Hard drive performance depends on movement, rotation, transfer, and cache

HDDs are mechanical devices, so their speed is affected by how quickly the drive can reach the data and how fast it can transfer the data once it is found.

Seek time head moves to the track Spindle speed platters rotate DTR data moves once found Cache recently used data is buffered
Seek time The time taken for the read/write head to move to the track where the data is stored. Lower seek time means less waiting.
Rotational latency and spindle speed The wait for the right sector to rotate under the head. Faster RPM can reduce this delay, though it can increase heat, noise, and power use.
Data transfer rate How quickly data can be read from or written to the drive once the drive has reached it, often measured in MB/s.
Cache Small fast memory inside the drive that stores recently used or predicted data to reduce repeated slow disk access.

Why this matters in a scenario

An HDD may be suitable for a large archive because capacity and cost matter most. It may be less suitable as a main editing drive because seek time and transfer limits can slow loading, saving, and working with large files.

Fragmentation

Fragmentation can slow hard disks because file parts are spread out

When files are created, edited, deleted, and saved over time, a file may be stored in separate pieces rather than one continuous run of space.

Fragmented file

A B A C A D

The HDD head may need to move between different areas of the platter to read all parts of file A.

After defragmentation

A A A B C D

Defragmentation rearranges file parts so related blocks sit closer together, reducing mechanical movement on an HDD.

Idea Meaning Exam point
Fragmentation One file is stored in separate non-contiguous blocks. Can increase seek time on HDDs.
Defragmentation Reorganises file blocks to make access more sequential. Can improve HDD performance after heavy file changes.
SSD difference Flash cells are accessed electronically, not by a moving head. Defragmenting an SSD is usually unnecessary and can add writes.

Exam Traps

Common mistakes with hardware devices

These questions often go wrong when students name a device but do not connect it to the scenario.

Confusing input and output

Touchscreens are often treated as only output devices.

Fix: Remember that some hardware performs more than one role.

Choosing by popularity

"Everyone uses it" is not a technical justification.

Fix: Mention speed, quality, portability, accessibility, capacity, or cost.

Treating portable storage as a backup plan on its own

Removable storage can help, but it may still be lost, stolen, or damaged.

Fix: Think about reliability as well as convenience.

Only saying "faster"

Storage answers often lose marks when they do not explain why one technology performs differently.

Fix: Mention moving parts, seek time, transfer rate, cache, or flash memory where relevant.

Defragmenting the wrong device

Defragmentation is mainly useful for HDDs because they have a moving head.

Fix: Do not recommend defragmenting an SSD as a normal performance improvement.

Question 1

Explain why a touchscreen may be more suitable than a keyboard and mouse in a public information kiosk.

Show answer guide
  • A touchscreen allows direct interaction with on-screen choices.
  • It is quicker and easier for short tasks than typing.
  • It reduces the number of separate devices needed.
  • It may feel more intuitive for casual users in public spaces.
6 marks Discuss

Question 2

A media student needs hardware for editing large video files while travelling between college and home. Discuss suitable storage choices for this situation.

Show answer guide
  • An SSD is a strong choice because it offers faster access and no moving parts.
  • No moving parts make it safer to carry while travelling.
  • Capacity still matters because video files can be very large.
  • A large HDD may be cheaper per gigabyte but is slower and more fragile.
  • A removable flash drive may help with transfer but may not be ideal as the main working drive.
  • The best answer balances portability, speed, and enough capacity for the project files.
8 marks Analyse

Question 3

A small design studio stores large project files on an older HDD. Staff say that files take longer to open after months of saving, deleting, and editing projects. Analyse possible HDD performance reasons and suggest suitable improvements.

Show answer guide
  • Large project files can be slow on an HDD because the drive has mechanical seek time and rotational delay.
  • Data transfer rate affects how quickly large files are read once the drive reaches the data.
  • Cache can help repeated access, but it cannot remove the limits of moving parts.
  • Frequent saving, deleting, and editing can cause fragmentation, making the head move to several places.
  • Defragmenting may improve HDD access if fragmentation is a problem.
  • Moving active project files to an SSD could improve access time and responsiveness because SSDs use flash memory with no moving parts.

Glossary

Quick glossary

Input device
Hardware used to send data into the system.
Output device
Hardware used to present results from the system.
Storage device
Hardware used to keep data for later use.
SSD
Fast solid-state storage with no moving parts.
HDD
Magnetic storage that uses spinning platters and moving heads.
Flash memory
Storage that keeps data using electrical charge in memory cells.
Seek time
Time taken for an HDD read/write head to reach the required track.
Data transfer rate
How quickly data can be read from or written to storage.
Fragmentation
When a file is stored in separate pieces across a disk.
