Overview

What you should know by the end of this lesson

Binary is the natural number system for computers because every bit is either 0 or 1. BCD keeps decimal digits separate, which makes some tasks easier but uses more storage.

Learning goals

  • Read binary values using place value.
  • Convert denary and binary accurately.
  • Explain what BCD stores and how it works.
  • Compare binary with BCD in context.

Quick memory hooks

  • Binary: base 2.
  • Denary: base 10.
  • BCD: each decimal digit uses 4 bits.
  • Nibble: 4 bits.

Where this shows up

Binary is used for direct machine representation. BCD is often chosen when systems need to store or display decimal digits in a form that maps cleanly to human-readable numbers.

Binary

Binary place value doubles as you move left

In binary, each column represents a power of 2. Starting from the right, the place values go 1, 2, 4, 8, 16, 32 and so on. A 1 means that value is included; a 0 means it is not.

8-bit example 128 64 32 16 8 4 2 1 Read from right smallest value first Add selected columns to get denary

Worked example

For 101101, the columns with 1s are 32, 8, 4, and 1. Add them: 32 + 8 + 4 + 1 = 45.

Conversion

Converting between denary and binary is about matching place values

To convert binary to denary, add the place values represented by the 1s. To convert denary to binary, subtract the largest possible powers of 2 until nothing is left.

Binary to denary Read each 1 bit as an included power of 2, then total them.
Denary to binary Mark a 1 where the power of 2 fits, and a 0 where it does not.
Example 13 becomes 1101 because 8 + 4 + 1 = 13.
Zero padding Leading zeros do not change the value but can keep a fixed bit width such as 8 bits.

BCD

Binary coded decimal stores each decimal digit separately

BCD does not convert the whole decimal number into one pure binary value. Instead, it converts each decimal digit into its own 4-bit binary pattern.

59 in BCD 0101 1001 5 becomes 0101 9 becomes 1001

Why BCD is useful

BCD can make decimal display and decimal-based data entry simpler because each digit remains separate. The trade-off is efficiency: it uses more bits than a pure binary value for the same number.

Exam Traps

Common mistakes with binary and BCD

Treating BCD as one pure binary number

In BCD, each decimal digit is handled separately. You should not read the full bit string as one normal binary value.

Fix: Split the BCD value into 4-bit groups first.

Skipping place values

Binary columns must follow powers of 2 in order.

Fix: Write the place values above the bits when working a calculation out.

Ignoring leading zeros in fixed-width questions

The value may be unchanged, but the required bit width still matters.

Fix: Match the number of bits requested in the question.

Glossary

Quick glossary

Binary
A base-2 number system using 0 and 1.
Denary
The usual base-10 number system.
BCD
Binary coded decimal, with each digit stored separately.
Nibble
A group of 4 bits.

Remember

Whole value or separate digits?

  • Pure binary converts the whole number.
  • BCD converts each decimal digit separately.
  • Place value is the key to both reading and building binary.
